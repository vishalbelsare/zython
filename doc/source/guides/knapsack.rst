Knapsack Problem
================

`Knapsack problem <https://en.wikipedia.org/wiki/Knapsack_problem>`_ is a famous problem in informatics.
There are several variations of the problem. In menu example we will look at the Knapsack with returns, were
you can order several portions of the same dish or do not order some menu's positions.
